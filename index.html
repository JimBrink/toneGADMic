This folder contains some p5.js files that might be of interest. Included are some possibly useful programs, some demo files, and some some how-to files related to p5.js.

The p5.js files have the file type .pjs. My editor has special features for different languages. It determines the language based on the file type which makes it desirable to distinguish between true Javascript .js files and p5.js files. In order to do so, the file type .pjs is being used for p5.js files.

The .pjs files assume the necessary p5.js include files are in "/p5js". If you decide to copy the files to your own computer, you may have to edit the names of those files in the <script> section.

Not many options at this point. Some more files should be coming soon.

  • Graphing converging Fourier Sine Series Helps illustrate the convergence of Fourier Series by graphing partial series sums for three common series.
  • "How-To" Files relating to using p5.js

The following two programs have been tested with Firefox, Chrome, Opera and the new Edge browsers. They may not work with the Edge Legacy browser. They have not been tested with Safari. The are not intended to be used and have not been tested on smart phones.

  • Tone Generator and Display (version 2.1.5) A tool that can generate sounds and display their wave form and harmonics. The help file contains a tour of the controls, discussion of the importance of harmonics and their use, and some experiments.

  • Tone Generator and Display with Mic (version 2.1.5) This the same as the above but it also allows microphone input. You will need a microphone attached to your computer to take advantage of this added feature. Its help file
